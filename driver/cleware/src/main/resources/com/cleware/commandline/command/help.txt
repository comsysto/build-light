

Cleware Traffic Light Java Driver, Maintained by B. Zuther.
Send bug reports using https://github.com/zutherb/build-light/issues

Usage: cleware [--gui] [--knightrider] [color on|off] [wait time]
  Default action is that the JavaFX GUI will be open;

  --gui          opens JavaFX GUI
  --knightrider  plays Knight Rider sequence on the traffic light
  color on|off   switchs Led on or off
  wait time      waits defined time in ms before the next action is executed

Examples:
  cleware --gui                     => opens JavaFX GUI
  cleware --knightrider             => plays Knight Rider sequence
  cleware red on wait 500 red off   => switch on red led, wait 500 ms, switch off red led
  cleware red on yellow on green on => switch on all leds
